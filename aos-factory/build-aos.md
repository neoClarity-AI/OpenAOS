# Build AOS — Entry Point

This is the AOS Factory's root entry file (§8.3). To build an Agentic Operating System instance, follow the master AOS builder:

```text
/builders/build-aos.md
```

To add a single agent to an existing AOS instance, use the generic agent build engine:

```text
/builders/build-agent.md
```

No files are created until the user types exactly `Proceed` (§3.1).
